About The Plugin

WooCommerce Discord Alerts is a lightweight WordPress plugin that sends real-time order notifications directly to your Discord server. Perfect for store owners and teams who want instant updates without logging into WordPress.

Features

✔ New Order Alerts
✔ Order Status Notifications
✔ Clean Discord Embeds
✔ Simple Setup
✔ Fast and Lightweight
✔ Works with All WooCommerce Stores

Installation Guide

1. Download plugin ZIP file

2. Go to WordPress Admin → Plugins → Add New

3. Upload Plugin ZIP

4. Activate Plugin

5. Activate admin page using license key

6. Paste Discord Webhook

7. Enable message alerts

8. Done ✅
