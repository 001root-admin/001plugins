About The Plugin

WooCommerce Discord Alerts is a lightweight plugin that sends real-time order notifications directly to your Discord server. Perfect for store owners and teams who want instant updates.

Features

✔ New Order Alerts
✔ Order Status Notifications
✔ Clean Discord Embeds
✔ Simple Setup
✔ Fast and Lightweight
✔ Works with All WooCommerce Stores

Installation Guide

1. Download plugin ZIP file

2. Go to WordPress Admin → Plugins → Add New

3. Upload Plugin ZIP

4. Activate Plugin

5. Paste Discord Webhook URL

6. Done ✅
